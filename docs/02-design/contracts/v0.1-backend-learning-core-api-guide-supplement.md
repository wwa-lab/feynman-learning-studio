## 1. Fixed Local URLs

- Swagger UI: `http://localhost:8080/swagger-ui/index.html`
- OpenAPI JSON: `http://localhost:8080/v3/api-docs`

## 2. Common Response Examples

Success:

```json
{
  "success": true,
  "data": {},
  "errorCode": null,
  "errorMessage": null
}
```

Paginated:

```json
{
  "success": true,
  "data": {
    "records": [],
    "pageNo": 1,
    "pageSize": 10,
    "total": 0
  },
  "errorCode": null,
  "errorMessage": null
}
```

## 3. Full Error Code List

- `VALIDATION_ERROR`
- `TOPIC_NOT_FOUND`
- `TOPIC_IN_USE`
- `EXPERIMENT_NOT_FOUND`
- `DUPLICATE_TOPIC_SLUG`
- `INVALID_TOPIC_REFERENCE`
- `INVALID_EXPERIMENT_STATUS`
- `INVALID_STATUS_TRANSITION`
- `DATABASE_ERROR`
- `SYSTEM_ERROR`

| Error Code | HTTP Status |
|---|---:|
